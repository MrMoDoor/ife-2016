# IFE-2016 前端泡沫组
我们组的成员都是来自南方的某一所大学的大二学生。专业有计算机，电子通信。这次来参加百度前端技术学院的课程，希望能够学到前端更加系统的知识。

我们队的名字叫做“前端泡沫组”。这名字颇有一种日本上世纪80年代末期的苍凉感。在世界资本主义的残酷中，我们希望能不像泡沫一样，在经济紧缩的时候破灭，却也希望能像泡沫一样，自由，轻盈，无所拘束，无所畏惧，无所谓。

[团队主页及demo](http://bubble.gq)
